# Prism Intelligence

AI-driven portfolio intelligence and performance analysis platform.

## Vision

Prism Intelligence measures whether real wealth was created beyond nominal market growth and cash inflows by comparing portfolio performance against Gold, Bitcoin, and inflation-adjusted USD benchmarks.

## Architecture

Clean Architecture (Ports & Adapters)

Lightweight DDD with Bounded Contexts

Plugin-based market extensibility

Independent engines for ingestion, backtesting, portfolio, and risk

## Tech Stack

Python 3.12+

FastAPI

PostgreSQL

Docker

SQLAlchemy 2.0

Redis
