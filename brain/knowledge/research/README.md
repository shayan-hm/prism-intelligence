# Research

This directory stores research documents, literature reviews, and deep-dive analyses.

## Purpose

- Store external research and internal investigations
- Keep technical deep-dives organized
- Reference from ADRs and domain docs

## Structure

```
research/
├── topic-name/
│   ├── README.md
│   ├── sources.md
│   └── analysis.md
└── ...
```

## Guidelines

- One folder per research topic
- Include `sources.md` with links and references
- Summarize findings in `analysis.md`
- Link from relevant ADRs using `References` section
